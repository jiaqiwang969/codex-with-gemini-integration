# Hunyuan AI3D 默认设置说明

## 🎯 当前默认配置

根据您的建议，我们已经优化了默认设置：

### 1. **默认 API 版本：Professional (专业版)**
- **原因**：Pro API 功能更强大，参数控制更精细
- **特点**：
  - 支持 PBR 材质生成
  - 支持面数控制（40K-1.5M）
  - 支持多种生成模式（Normal/LowPoly/Geometry/Sketch）
  - 支持多视角输入
  - 生成质量更高

### 2. **默认输出格式：OBJ**
- **原因**：OBJ 是最通用的3D格式
- **优势**：
  - ✅ 所有3D软件都支持
  - ✅ 包含材质文件（.mtl）
  - ✅ 包含纹理贴图（.png）
  - ✅ 文件结构清晰

## 📝 使用示例

### 最简单的使用（使用所有默认值）

```
生成一个可爱的机器人3D模型
```
- 自动使用 Professional API

## 🔧 API 版本对比

| 特性 | Professional API（默认） | Rapid API |
|------|------------------------|-----------|
| **速度** | 较慢（2-5分钟） | 快（30-60秒） |
| **输出格式** | 自动选择最优格式 | OBJ（固定） |
| **质量** | 高质量，细节丰富 | 良好 |
| **PBR材质** | ✅ 支持 | ❌ 不支持 |
| **面数控制** | ✅ 40K-1.5M可调 | ❌ 固定 |
| **生成模式** | ✅ 4种模式 | ⚠️ 基础模式 |
| **多视角输入** | ✅ 支持 | ❌ 不支持 |
| **Sketch模式** | ✅ 文字+图片 | ❌ 不支持 |
| **适用场景** | 最终产品、高质量需求 | 快速原型、测试 |

## 📦 输出文件结构（OBJ格式）

使用默认设置后，您会得到：
```
outputs/hunyuan/
├── {job_id}_obj.zip          # 压缩包（自动解压）
├── {model_id}.obj            # 3D模型文件
├── material.mtl              # 材质定义
└── material.png              # 纹理贴图
```

## 💡 最佳实践

1. **快速测试**：使用默认的 Rapid API + OBJ
2. **高质量需求**：明确指定 Professional API
3. **特定格式需求**：使用 Rapid API（仅支持 OBJ）

## 🚀 为什么选择这些默认值？

1. **Professional API（专业版）**
   - 功能最全面，参数控制精细
   - 支持所有高级特性（PBR、面数、多视角等）
   - 生成质量更高，适合专业需求
   - AI可以根据用户需求智能调整参数

2. **OBJ 格式**
   - 行业标准，兼容性最好
   - 文本格式，易于调试
   - 支持完整的材质系统
   - 所有3D软件都能打开

3. **自动化流程**
   - 减少用户配置
   - 开箱即用
   - 智能默认值
   - 一站式完成（提交→轮询→下载）

现在您只需要简单地描述想要的3D模型，系统会自动处理所有技术细节！
