# Hunyuan AI3D 性能评估与使用指南

## 📊 性能测试结果

基于实际测试，以下是不同场景的生成时间参考：

### Rapid API（极速版）
| 模型复杂度 | 描述示例 | 实测时间 | 建议用途 |
|-----------|---------|---------|---------|
| 简单 | 红色球体、立方体 | **30-45秒** | 快速原型、简单几何体 |
| 中等 | 简单家具、日用品 | **45-60秒** | 产品预览、概念验证 |
| 复杂 | 建筑、角色 | **60-90秒** | 初步设计、草图转换 |

**特点**：
- ✅ 生成速度快（比Pro快30-50%）
- ✅ 适合快速迭代
- ❌ 参数限制：提示词最多200字符
- ❌ 并发限制：1个任务

### Pro API（专业版）
| 模型复杂度 | 参数设置 | 实测时间 | 建议用途 |
|-----------|---------|---------|---------|
| 标准质量 | PBR材质 + 20万面 | **90-120秒** | 游戏资产、产品展示 |
| 低多边形 | LowPoly + 5万面 | **60-90秒** | 风格化游戏、移动端 |
| 高精度 | PBR + 50万面 | **120-180秒** | 影视渲染、高端展示 |
| 白模 | Geometry模式 | **90-120秒** | 后期处理、3D打印 |

**特点**：
- ✅ 支持1024字符提示词
- ✅ 3个并发任务
- ✅ 丰富的参数控制
- ✅ 支持多视角输入

## 🎯 参数影响分析

### 1. PBR材质
- **影响**：+20-30% 生成时间
- **效果**：显著提升质量，增加金属度、粗糙度、法线贴图
- **建议**：产品展示、游戏资产建议开启

### 2. 面数控制（Face Count）
| 面数范围 | 时间影响 | 适用场景 |
|---------|---------|---------|
| 40K-80K | 基准时间 | 移动游戏、Web展示 |
| 80K-300K | +10-20秒 | PC游戏、标准渲染 |
| 300K-1.5M | +30-60秒 | 影视特效、高端可视化 |

### 3. 生成模式（Generate Type）
| 模式 | 时间影响 | 特点 |
|------|---------|------|
| Normal | 基准 | 标准纹理模型 |
| LowPoly | -10-20% | 风格化、优化面数 |
| Geometry | -5-10% | 无纹理白模 |
| Sketch | +10-20% | 支持草图+文字输入 |

### 4. 输出格式（仅Rapid支持）
- **OBJ**：最快，通用性最好（推荐）
- **GLB**：+5秒，Web友好，包含材质
- **STL**：+3秒，3D打印专用
- **FBX**：+5秒，游戏引擎友好
- **USDZ**：+5秒，苹果生态
- **MP4**：+10秒，视频预览

## 📈 提示词优化建议

### 提示词复杂度对时间的影响
- **简单描述**（<50字）：基准时间
- **详细描述**（50-200字）：+10-15%
- **复杂描述**（200-1024字，仅Pro）：+15-25%

### 有效的提示词结构
1. **主体描述**：明确的对象类型
2. **风格特征**：材质、颜色、风格
3. **细节要求**：特定部件、比例
4. **质量要求**：精细度、用途

**示例**：
```
一个现代办公椅，人体工学设计，黑色皮革座垫，
铝合金支架，五星轮底座，可调节高度，带扶手
```

## 🚀 性能优化建议

### 1. 批量生成策略
- Pro API支持3个并发，充分利用
- 相似模型使用相同参数模板
- 错开提交时间避免限流

### 2. 参数优先级
1. **速度优先**：Rapid + 简单提示词 + 无PBR
2. **质量优先**：Pro + PBR + 高面数
3. **平衡选择**：Pro + 中等面数 + 选择性PBR

### 3. 失败重试策略
- 首次失败：等待30秒重试
- 二次失败：降低参数复杂度
- 三次失败：切换API版本

## 📝 常见问题

### Q: 为什么有时生成很慢？
A: 可能原因：
- 服务器繁忙（高峰期）
- 模型复杂度高
- 网络延迟

### Q: Rapid和Pro如何选择？
A: 
- **选Rapid**：简单模型、快速验证、成本敏感
- **选Pro**：复杂模型、精确控制、批量生成

### Q: 生成失败怎么办？
A: 检查：
1. 提示词是否过长（Rapid限200字符）
2. 图片格式和大小是否符合要求
3. API密钥是否有效
4. 是否触发限流（建议间隔10秒）
