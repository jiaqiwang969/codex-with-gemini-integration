
### 4. 高级参数使用

#### 示例1：生成低多边形模型
```
生成一个低多边形风格的树木，用于游戏场景
参数：LowPoly模式，三角形面，面数限制50000
```

对应的MCP调用：
```json
{
  "prompt": "一棵低多边形风格的树木",
  "generate_type": "LowPoly",
  "polygon_type": "triangle",
  "face_count": 50000
}
```

#### 示例2：生成高质量PBR材质模型
```
生成一个金属机器人，需要PBR材质
参数：启用PBR，面数150000
```

对应的MCP调用：
```json
{
  "prompt": "金属质感的机器人",
  "enable_pbr": true,
  "face_count": 150000
}
```

#### 示例3：生成白模（无纹理）
```
生成一个建筑的白模，用于建筑展示
参数：Geometry模式
```

对应的MCP调用：
```json
{
  "prompt": "现代建筑",
  "generate_type": "Geometry"
}
```

## 输出文件说明

生成完成后，您会在 `outputs/hunyuan/` 目录下找到：

```
outputs/hunyuan/
├── {job_id}_preview.png     # 3D模型预览图
├── {job_id}_obj.zip         # 压缩包（自动解压）
├── model.obj                # 3D模型文件
├── material.mtl             # 材质定义文件
└── texture.png              # 纹理贴图
```

### 支持的输出格式
- **OBJ**: 通用3D格式，兼容性最好
- **GLB**: 包含纹理的单文件格式
- **FBX**: 专业3D软件格式
- **USDZ**: Apple设备专用格式

## 常见问题解答

### Q: 生成需要多长时间？
A: 通常30-60秒，复杂模型可能需要2-3分钟。

### Q: 如何提高生成质量？
A: 
1. 提供详细的文字描述
2. 使用高质量的参考图片
3. 启用PBR材质
4. 增加面数限制

### Q: Sketch模式什么时候使用？
A: 当您有参考图片，但还想通过文字描述添加或修改细节时。

### Q: 如何处理生成失败？
A: 系统会自动重试，如果仍然失败，请检查：
1. 图片格式是否正确（支持jpg/png/webp）
2. 图片大小是否超限（建议<6MB）
3. 文字描述是否过长（限制1024字符）
